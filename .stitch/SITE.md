# Beanstalk Coffee — Site Vision

## 1. Vision
A premium, nature-boutique landing page for Beanstalk Coffee, Jaipur. The site delivers an Awwwards-level sensory experience combining a 3D scroll-animated coffee cup, particle physics, live utility, and layered parallax storytelling.

## 2. Tech Stack
- Next.js 16, Tailwind CSS v4, Framer Motion, Zustand
- Hosted: Vercel (target)

## 3. Design Theme
Nature-Boutique Luxury. Espresso/Forest Green/Parchment palette. Glassmorphism + parallax + boldly typographic.
